# 🏠 House Rental System

A full-featured house rental web application built with Django, allowing landlords to list properties and tenants to book them online.

## 🔗 Live Demo
Coming soon

## ✨ Features

### For Tenants
- Browse and search available properties
- Filter by city, property type, and rent range
- Submit booking requests
- Make payments online
- Submit maintenance requests
- Leave reviews for properties

### For Landlords
- Add, edit, and delete property listings
- Upload property images
- Approve or reject booking requests
- Track revenue and payments
- Manage maintenance requests

### For Admin
- Manage all users, properties, and bookings
- Approve new user accounts
- View system-wide statistics

## 🛠 Tech Stack
- **Backend:** Python, Django 6.0
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Authentication:** Django Auth

## 🚀 Getting Started
```bash
git clone https://github.com/Bkash123/house-rental-system.git
cd house-rental-system/house_rental_system
pip install django
python manage.py migrate
python manage.py runserver
```
